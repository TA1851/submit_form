### Overview

#### main/sample.xlsm

UserForm に各Labelに値を入力し、入力した値をセルに書き込む。

* 各 Label
  
  * Date
  * 件名
  * 作業工数(Is)
  * 作業工数(Was)
  * ww件数
  * 工数（人）
  
* Button

  * Clear -> 入力した値の削除
  * 登録  -> 入力した値をセルに一括入力する  
  * 閉じる -> Form を閉じる
